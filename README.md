# wakadisplay

This project is based off of the Spotify Display starter project.

It shows today's and the week's coding statistics, fetched from the WakaTime API.

I have logged over 900 hours on WakaTime over the few years since I began using it to track my programming. I find having metrics such as WakaTime extremely useful, but it can be tedious to have to constantly check. Having a physical presence that reminds you of how long you've been working on something, and whether it might be time to take a break, would be great. Hence: wakadisplay!
